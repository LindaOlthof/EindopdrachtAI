Bibliotheek systeem voor bordspellen

-> Bordspellenregistratie (feature 1)
    - Spellen toevoegen
    - Spellen verwijderen
    - Bordspel gegevens aanpassen
    - Alle spellen bekijken

-> Speelregistratie (feauture 2)
    - Gespeeld spel
    - Spelers
    - Winnaar
    - Score (indien van toepassing)
    - Duur spel

-> Statistieken (feature 3)
    - Meest gespeelde spel (& Gameplay)
    - Topspeler van het spel
    - Gemiddelde spelduur

-> Filter/zoekfunctie indien tijd dat toelaat (feature 4)
    - Naam spel
    - Duur spel
    - Aantal spelers
    - Speelniveau
    - Gameplay
    - Populariteit
